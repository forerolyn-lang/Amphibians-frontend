🧠 Introducción 01

PROBLEMA

La vigilancia epidemiológica del COVID-19 requiere clasificar casos de forma eficiente. El dataset Surveillance.csv contiene 7 variables categóricas (A01–A07) con signos y síntomas que permiten predecir la categoría clínica: PUS, PIM o PWS.

Este proyecto entrena y compara una Red Neuronal (MLP) y Regresión Logística, con predictor interactivo que acepta el dataset real para calcular la matriz de confusión.

ESPECIFICACIONES

  • DatasetSurveillance.csv
  • VariablesA01 – A07 (+/-)
  • TargetCategories (PUS/PIM/PWS)
  • TareaClasificación multiclase
  • Modelo AMLPClassifier
  • Modelo BLogisticRegression
  • EscaladoStandardScaler
  • DeployGitHub Pages
